As a customer I should be able to see a list of items that are available to rent.
As a customer I should be able to filter that list of items based on attributes of items.
As a customer I should be able to view a list of standard attributes for every item.  The item  should have an attributes of Cost Per Day, Minimum Day Rental, Category, Array of images, Description, Title, Deposit Amount, User Requirements.
As I user I should be able to see a calendar which shows the state of each item.  The states resolve to a specific date so each date has a single status.  The possible states are "Out for Use", "Booked", "Open for Booking", “Readying for Use”.
As a user I should be able to request an item to be booked for days that are currently statused as “Open for Booking”.
As a user I should be able to request a booking for any set of days within the next 60 days for any item that is in an “Open for Booking” state for those days.
As a user I should receive discount for the item rental based on the duration of the rental period.  The discount should be 5% per day up to 15%.

As an admin of the system I should be able to add items for rental.
As an admin of the system I should be able to add the attributes for the items.
As an admin of the system I should be able to accept a proposed booking.
As an admin of the system I should be able to decline a proposed booking by entering a reason, have the customer notified by email with the item and requested rental period, and have those dates marked open for booking again.
As an admin of the system I should be able update the status of each item.
As an admin of the system I should be able to hide an item from the public catalog while still seeing it in the admin item list, with inactive items clearly highlighted.

As a user I should be able to select the Category when filtering from all the distinct categories of existing items.
As a user I should be able to set minimum and maximum $ / day using a single range control with a slider on each end (whole dollars).
As a user I should be able to filter on items that are "Open for Bookings" during a selected date range in combination with the category.

As a user when I am requesting a booking I should be able to upload a picture of my drivers license.
As a user I should also be able to upload a picture of my license plate for any items that are tagged as towable.
As an admin I should be able to mark an item as towable.

As a user when I request a booking I should be required to provide an email address and a phone number.
As a user I should receive the rental quote at my email address when I request a quote (when email delivery is configured on the server).
As a user I should see first name, last name, and address fields when requesting a booking, and the system should attempt to extract those from my driver license photo so I can confirm or correct them before submitting (preferably using a local model so my license image is not sent to a third party).

Use Auth0 for user authentication but stub it out for now
